# Welcome to [Slidev](https://github.com/slidevjs/slidev)!

To start the slide show:

- `pnpm install`
- `pnpm dev`
- visit <http://localhost:3030>

Edit the [slides/slides.md](./slides/slides.md) to see the changes.

Learn more about Slidev at the [documentation](https://sli.dev/).

- 初回のみ必要: `npm i -D playwright-chromium`
- PDF出力コマンド: `npx slidev export slides.md --with-clicks`
- 補足: PDF出力のために `slides/slides.md` を毎回編集する必要はありません。

###
1. テクノロジ系：基礎理論とアルゴリズム
ITの仕組みやプログラミングの基礎的な考え方を問う分野で、2次式の知識が顔を出します。
処理回数（計算量）の見積もり: プログラムの中で「ループ（繰り返し）処理の中に、さらにループ処理がある」という二重ループ構造のアルゴリズムを実行した場合、データ量 $n$ に対して処理時間はおよそ $n^2$（$n$の2乗）に比例して増加します。この「データが増えると処理時間が2次関数的に急増する」という感覚を理解しておく必要があります。
組み合わせの計算: 例えば「ネットワーク上にある $n$ 台のパソコンすべてを、互いに1対1でケーブル接続する場合、ケーブルは何本必要か？」といった問題です。この計算式は $\frac{n(n-1)}{2}$ となり、展開すると $n^2$ を含む2次式になります。

2. ストラテジ系：経営科学・データ分析
会社の経営戦略や利益を計算する分野で、グラフや最適化の問題として登場することがあります。
利益の最大化（グラフの読み取り）: 商品の価格設定と売上予測から利益を求める際、グラフが「上に凸の放物線（山の形をした2次関数のグラフ）」になることがあります。この場合、「グラフの頂点＝利益が最大になる価格」という関係を読み解く力が求められます。


試験対策としてのアドバイス
ITパスポートにおける数学は、あくまで「ITやビジネスの課題を解決するための道具」として出題されます。そのため、以下のように割り切って対策することをおすすめします。
1. 「解の公式」や「因数分解」の丸暗記は不要: 複雑な計算を自力で解く必要はほぼありません。
2. 選択肢を「代入」して解く: もし2次方程式を解くような場面に遭遇しても、ITパスポートは四肢択一のマークシート形式です。「ア、イ、ウ、エ」の選択肢の数字を式に当てはめてみて、計算が合うものを探すというアプローチで確実に正解できます。
3. グラフの形と意味を紐づける: 式の計算よりも、「このグラフ（放物線）の頂点は何を意味しているのか？」といった、視覚的・論理的な理解を深めることに時間をかけましょう。
